# AI News Summarizer

An intelligent n8n automation workflow that gathers AI news, technology updates, and upcoming AI events from multiple sources, summarizes them using Gemini AI, and delivers a personalized newsletter directly to email.

## Features

* Automated daily execution using Schedule Trigger
* AI news aggregation from RSS feeds
* Technology news aggregation from RSS feeds
* Real-time AI event discovery using SERP API
* Data aggregation from multiple sources
* AI-powered summarization using Gemini AI
* Automated newsletter generation
* Email delivery using Gmail

## Workflow Architecture

1. Schedule Trigger
2. Fetch AI News RSS Feeds
3. Fetch Technology RSS Feeds
4. Search AI Events using SERP API
5. Merge Data Sources
6. Aggregate Content
7. Generate Summary using Gemini AI
8. Send Newsletter via Gmail

## Technologies Used

* n8n
* Gemini AI
* Gmail API
* RSS Feeds
* SERP API

## Problem Solved

Instead of spending 30+ minutes browsing multiple websites for AI news, technology updates, and upcoming events, users receive a concise personalized newsletter in their inbox within minutes.
